# Setup

1. Clone the repository
2. Run `npm install`

# Running the app

## Development mode

```bash
npm run dev
```

The site will be available at `localhost:5600`

## Production mode

```bash
npm run build
npx http-server build -p 4200 -a localhost
```

# Linting

View linting errors by running:

```
npm run lint
```

# Flow

View flow errors by running:

```
npm run flow
```
